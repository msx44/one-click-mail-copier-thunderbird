# 📋 One-Click Mail Copier for Thunderbird

<img src="icon.png" width="120" alt="One-Click Mail Copier Icon" align="right">

Thunderbirdで表示しているメールの情報を、ワンクリックでお好みのフォーマットに整形してクリップボードにコピーできる実務特化型アドオンです。

日本のビジネスメール環境にありがちな「特定の文字コードによる文字化け（Latin-1誤認問題）」を自動で検知して強制救済する、鉄壁のデコードロジックを搭載しています。

---

## ✨ 主な機能

* **ワンクリック一発コピー**: 3ペイン画面や独立ウィンドウから、ボタン一つで瞬時にデータを取得。
* **自由なマクロテンプレート**: 管理画面から、好みの並び順や独自のフォーマット（社内システム貼付用など）を自由にカスタマイズ可能。
* **文字化け自動撲滅**: `iso-2022-jp` や `shift_jis`、特定のUTF-8生バイトが誤認識されて「ã‚ˆã‚Š」のように化ける現象を自動で判別・復元。
* **完全ローカル処理**: 取得したメールデータは一切外部のサーバーに送信されないため、セキュリティも安心。

---

## ⚙️ 利用可能なマクロタグ

アドオンの設定画面（アドオンマネージャー ＞ 歯車マーク）から、以下のタグを組み合わせた自由なテンプレートを作成できます。

| タグ | 出力される内容 |
| :--- | :--- |
| `{{subject}}` | メールの件名 |
| `{{date}}` | 送信日時（`YYYY/MM/DD HH:mm:ss` 形式） |
| `{{from}}` | 送信者（Fromのメールアドレス・名前） |
| `{{to}}` | 宛先（To）※複数ある場合はカンマ区切り |
| `{{cc}}` | CC ※複数ある場合はカンマ区切り |
| `{{bcc}}` | BCC ※複数ある場合はカンマ区切り |
| `{{id}}` | Thunderbird内部のメールID |
| `{{body}}` | メールの本文（プレーンテキスト） |

### 💡 テンプレート設定例（デフォルト）
```text
件名: {{subject}}
Date: {{date}}
From: {{from}}

{{body}}
```
## 🚀 インストール方法
公式ストアからインストール（推奨）
現在Thunderbird Add-onsストアにて審査中です。公開され次第、こちらにリンクが掲載されます！

### 手動インストール（開発者向け / 身内配布用）
本リポジトリのソースコード、またはリリースから .xpi ファイルをダウンロードします。

Thunderbirdの「アドオンマネージャー」を開きます。

歯車アイコンから「ファイルからアドオンをインストール」を選択し、読み込ませてください。
